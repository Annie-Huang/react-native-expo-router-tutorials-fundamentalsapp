- cd react-native-expo-router-tutorials-fundamentalsapp

Do you want to move existing files to /app-example instead of deleting them? (Y/n): n

PS C:\react\react-native-expo-router-tutorials-fundamentalsapp> npx expo start

Catch-all segments
[..param].tsx
e.g.
app/products/[...rest].tsx


http://localhost:8081/
http://localhost:8081/about
http://localhost:8081/profile
http://localhost:8082/products
http://localhost:8082/products/3

(below require more than 1 folder, if it is just 1 folder deaph, it will still go to app/products/[id].tsx file)
http://localhost:8082/products/best-sellers/playstation-5               << Catch-all segments: use app/products/[...rest].tsx
http://localhost:8082/products/best-deals/black-friday/playstation-5    << Catch-all segments: use app/products/[...rest].tsx
http://localhost:8082/products/search/playstation-5                     << Catch-all segments: use app/products/[...rest].tsx

http://localhost:8082/missing-route     << if 'app/+not-found.tsx' is there, go display this page.
                                            Otherwise, go to a builtin default 'Unmatched Route' page from expo-router.
                                            This is more relevant to webpage, as in mobile app you can only navigate through <Link> in the app.
http://localhost:8082/_sitemap          << site map
